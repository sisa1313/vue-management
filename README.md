### 安装运行
##### 1.下载或克隆项目源码
##### 2.npm 或者 cnpm安装相关包文件(推荐使用cnpm)

```js
npm or cnpm i
```
##### 3.启动项目
```js
npm run dev
```
##### 4.打包项目
```js
npm run build
```
